## About This Session

This is your first session and it is all about getting set up and ready for the 6-month journey ahead.

You will be introduced to the MNSUG mentorship community, the SC-500 exam structure, and how the Saturday sessions will run. You will also set up your GitHub account, fork this repository, and learn how to submit your notes after each session.

**By the end of this session you will be able to:**
- Understand the SC-500 certification and why it matters for cloud security careers
- Know what to expect each Saturday for the next 6 months
- Have your GitHub account set up and this repo forked
- Know how to download a template, fill it in, and upload it back

## How to Submit Your Notes

> You do not need to know git commands. Just follow these 5 steps.

1. **Fork** this repository — click the **Fork** button at the top-right of the [main repo page](https://github.com/Microsoft-Naija-Security-Usergroup/github-onboarding-workshop)
2. **Navigate** into `classes/session-00-2026-06-27-pre-bootcamp-onboarding/` on your forked repo
3. **Download** the template file `pre-bootcamp-onboarding-yourname-template.md` — click it, then click the **Raw** button, then right-click and choose *Save As*
4. **Rename** the file on your computer — replace `yourname` with your actual name in lowercase with hyphens, e.g. `pre-bootcamp-onboarding-firstname-lastname.md`
5. **Fill in your notes** using any text editor (Notepad, VS Code, TextEdit)
6. **Upload** your completed file back to GitHub — go into `classes/session-00-2026-06-27-pre-bootcamp-onboarding/` on your fork, click **Add file → Upload files**, drag in your file, then click **Commit changes**
7. **Open a Pull Request** — the facilitator will review your notes before merging
